## Гайд по запуску
### 1. Установка пакетов
```bash
yarn install
```

### 2. Запуск проекта 
```bash
yarn dev
```

### 3. Перейти по адресу
### http://localhost:3000

## Гайд по генерации и запуска докер контейнера

### 1. Генерация image и создание и запуск контейнера
```bash
sudo docker compose up -d
```

### 2. Отключение контейнера
```bash
sudo docker compose down
```
### 3. Удаление image
```bash
sudo docker rmi nextjs:v3
```
